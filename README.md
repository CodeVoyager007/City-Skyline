## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/CodeVoyager007/city-skyline.git
   ```
2. Navigate to the project directory:
   ```sh
   cd city-skyline
   ```

Open `index.html` in your favorite web browser to view the city skyline.
